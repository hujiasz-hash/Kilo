# 营养估算参考

## 常见食物基准值

| 食物 | 每100g估算 |
|------|-----------|
| 米饭 | 116kcal, 碳水25g, 蛋白2.6g, 脂肪0.3g, 纤维0.4g |
| 炒菜类 | 额外 +5-10g 脂肪（约45-90kcal） |
| 汤类 | 清汤约25kcal，浓汤约55kcal+ |
| 外卖 | 比家常菜油多 30-50% |
| 水果 | 100g 约 30-60kcal |
| 含糖饮料 | 约 10g糖/100ml |

## 份量换算

| 描述 | 估算克数 |
|------|---------|
| 一碗米饭 | ≈200g |
| 一个鸡蛋 | ≈50g |
| 一杯牛奶 | ≈250ml |
| 一个馒头 | ≈100g |
| 饭碗大小 | ≈200g 米饭 |
| 餐盘大小 | ≈300-400g 菜肴 |

## 代餐/补剂/饮料

- **蛋白粉**：每 scoop ≈ 30g ≈ 120kcal, 蛋白24g
- **代餐奶昔**：每瓶 250-400kcal
- **奶茶/咖啡**：大杯约 350-500kcal（含珍珠等配料）
- **啤酒**：100ml ≈ 43kcal
- **红酒**：100ml ≈ 85kcal
- **白酒**：100ml ≈ 298kcal（通常小杯50ml）

## 复合食物拆分

- 饺子/馄饨 → 面皮 + 馅料分别估算
- 汉堡 → 面包 + 肉饼 + 酱料分别估算
- 炒饭 → 米饭 + 鸡蛋 + 油分别估算
- 披萨 → 面饼 + 奶酪 + 配料分别估算

## 准确度标注

不确定时在回复中标注 "⚠️ 估算值，实际可能有 ±15% 误差"。
外卖估算误差更大，建议标注 "预估来自外卖订单，实际可能有差异"。
