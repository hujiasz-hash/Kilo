# 输出格式规范

## 食物识别输出

```
🍽️ {餐别}

{食物名} 约{份量}g — {热量}kcal
├ 碳水 {c}g  蛋白 {p}g  脂肪 {f}g  纤维 {fiber}g

{多种食物逐行列出来}

─────────────
📊 本餐合计：{热量}kcal | 碳水{总量}g 蛋白{总量}g 脂肪{总量}g 纤维{总量}g

📋 今日进度（vs 设定目标）
热量  {今日热量}/{目标}kcal  {progress_bar} {pct}%
碳水  {今日碳水}/{目标}g     {progress_bar} {pct}%  {over_text}
蛋白  {今日蛋白}/{目标}g     {progress_bar} {pct}%  {over_text}
脂肪  {今日脂肪}/{目标}g     {progress_bar} {pct}%  {over_text}
纤维  {今日纤维}/{目标}g     {progress_bar} {pct}%  {over_text}

🔖 #{meal_id}

{fiber_warning_if_needed}
```

## 进度条规则

用 █ 表示已完成，░ 表示剩余。10格总长度，按百分比填充。

```
热量  512/2000 kcal ███░░░░░░░ 25.6%
```

某项超过100%时标记为 🟡 黄字。

## 纤维预警

当日纤维摄入 < 50% 目标时出现：

```
💡 今天膳食纤维仅摄入 {纤维当前}g
   距推荐 {纤维目标}g 还差 {缺口}g
   建议：西兰花、燕麦、绿叶蔬菜或一把坚果
```

## 条码查询输出

**查到：**

```
📦 查到 {品牌} {食物名}
每100g：{calories}kcal | 碳水{carbs}g 蛋白{protein}g 脂肪{fat}g 纤维{fiber}g
每份({serving_desc})：{per_serving}kcal

要记录到这次用餐吗？
```

**未查到：**

```
未找到条码 {数字} 对应的食品。
您可以通过以下方式添加：
· 拍一张营养成分表照片，我自动读取
· 直接告诉我食物名称和营养数据
```

## 营养表扫描输出

```
📋 从营养成分表识别到：

{食品名称}
每100g：{calories}kcal | 碳水{carbs}g | 蛋白{protein}g | 脂肪{fat}g{fiber_text}

数据正确吗？
如正确，回复"确认"即可录入数据库
如需修改，回复"修改 {字段} {值}"
```

## 外卖订单输出

```
📱 识别到外卖订单：{店铺名}

{菜品1} ×{份数} → 约{热量}kcal
├ 碳水 {c}g  蛋白 {p}g  脂肪 {f}g  纤维 {fiber}g

{菜品2} ×{份数} → ...

─────────────
共 {n} 项菜品
回复"确认"全部记录，或指定记录哪些
```

## 体重输出

```
⚖️ 体重记录：{kg}kg
📊 较上次：{change}kg  |  📈 近7天：{trend}

确认记录体重吗？
```

如果数字相比上次变化 > 5kg，提示确认异常。
